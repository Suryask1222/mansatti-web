## 📖 About The Project

**Mansatti** is a modern e-commerce platform developed to showcase and sell authentic clay cookware, earthen pots, handcrafted products, and traditional village-made items online.

The platform combines modern web technologies with traditional craftsmanship, helping local artisans reach a wider audience while providing customers with a seamless shopping experience.
